# 🎨 Counter-Strike Skins Bot

Bot for tracking skin prices, auto-buying/selling on Steam Marketplace or third-party sites.

## Features
- 💰 Real-time price monitoring (Buff, Steam, Skinport)
- 🤖 Auto-trade with pre-set filters (float, pattern, stickers)
- 📊 Profit calculator & inventory history
- 🔔 Discord webhook notifications for good deals
